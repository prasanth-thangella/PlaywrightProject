# PlaywrightProject

This project contains Playwright automation scripts and examples.

## Structure
- `Chapter01/01_Verfiy.js`: Basic Node.js script to verify environment and Node version.

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/prasanth-thangella/PlaywrightProject.git
   ```
2. Install dependencies (if any):
   ```sh
   npm install
   ```
3. Run scripts:
   ```sh
   node Chapter01/01_Verfiy.js
   ```

## .gitignore
This project uses a `.gitignore` file to exclude common Node.js, Playwright, and environment files.
